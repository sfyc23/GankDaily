# 代码&妹子
这是一款Bottom Navigation guidelines风格的Gank的客户端。
使用了主流的RxJava+Retrofit+MVP构建，其中的api来源于 http:gank.io

------

##功能介绍##
每日干货查看，美女查看，视频查看等。
